# MyTunes  -   HTML / CSS | Responsive Landing Page 

MyuTunes - This is a responsive landing page It was be modeled after the iTunes 
website as far as layout and content. Used HTML5, CSS3 and a bit of jQuery.
 
Technology used:

CSS3
